# [기획서] 학년별 위험사고 예측 — 학교안전 가이드라인용 데이터 만들기

### 1. 주제 및 선정 배경
최근 5년간(2021–2025) 학교안전사고는 **865,384건**, 총 보상금 **약 2,357억 원**에 달하며,
사고는 2021년 대비 2025년 **2.36배 증가**하였다. 그러나 기존 안전통계는 **단순 발생 건수**만
보여줘, "자주 일어나는 사고"와 "크게 다치는 사고"를 구분하지 못해 예방 우선순위가 모호하다.

### 2. 목적
사고의 **빈도(건수)** 와 **심각도(보상금)** 를 하나로 결합한 **종합위험지수**를 정의하고,
이를 **학교급·학년·사고형태·장소** 단위로 산출하여 **학년별 안전 가이드라인용 데이터**를 만든다.

```
종합위험지수 = 사고건수(빈도) × 평균보상금(심각도)   ← 기대손실 개념
```

### 3. 데이터 및 방법
- **데이터**: 중앙회 제공 사고 데이터(86.5만 건)·보상 데이터(52.8만 건).
- **방법**:
  1) 연도별 시트 결합·전처리, 7개 급여 합산→총보상금(심각도).
  2) 사고(빈도)·보상(심각도)을 공통 키로 결합해 종합위험지수 산출.
  3) 빈도·심각도 **사분면**으로 위험 유형 분류.
  4) 학년별 상위 위험 시나리오 Top5 → **가이드라인 데이터(60행)** 생성.
- **도구**: Python(pandas·matplotlib·seaborn), 전 과정 로컬 수행·재현 가능.

### 4. 핵심 결과(요약)
- 빈도 1위는 `넘어짐`, 심각도 1위는 `익사·익수(평균 1.2억)`·`교통사고(1,345만)` 로 서로 다름.
- **종합위험지수 1위는 `넘어짐`(1,199억)** — 흔하면서 누적 피해도 최대 → 최우선 관리.
- 위험은 **운동장·강당(체육관)** 에 집중되며, **학년마다 위험 시나리오가 상이**함.
- 최종 산출: 초·중·고 **12개 학년 × Top5 = 60개** 위험 시나리오 데이터.

### 5. 기대효과 및 활용방안
- 학년별 "우리 학년 Top5 위험" **안전 가이드라인 카드/포스터** 자동 제작 근거.
- 종합위험지수 상위 시나리오에 **시설 투자·예방 자원 우선 배분**.
- 저빈도·고심각(익사·교통·추락)은 **중대사고 별도 관리군**으로 식별·대비.
- `recommend(학교급, 학년)` 함수로 안전관리 시스템·앱에 데이터 연계 확장 가능.
