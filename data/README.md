# 데이터 안내

## ⚠️ 원본 데이터는 이 저장소에 포함되지 않습니다

본 프로젝트는 **학교안전공제중앙회 제공 「2021–2025 학교안전사고 데이터」**를 분석합니다.
해당 원본 데이터는 경진대회 규정상 **외부 공개·재배포가 금지**되어 있어, 이 저장소에는
포함하지 않으며 `.gitignore` 로 제외되어 있습니다.

```
data/
├── raw/        # (비공개) 중앙회 제공 원본 xlsx 2개를 여기에 둡니다 — git 추적 제외
│   ├── ★2021-2025 학교안전사고 데이터.xlsx
│   └── ★2021-2025 학교안전사고 보상 데이터.xlsx
└── sample/     # (공개) 동일 스키마의 합성 샘플 — 코드 재현용
    ├── sample_accident.xlsx
    └── sample_compensation.xlsx
```

## 코드를 재현하려면 (원본 없이)

원본이 없어도 동일 스키마의 **합성 샘플 데이터**로 전체 파이프라인을 그대로 실행할 수 있습니다.

```bash
python scripts/make_sample_data.py   # 합성 샘플 생성 (난수 — 통계적 의미 없음)
python scripts/run_analysis.py       # 분석 실행 (raw 없으면 sample 자동 사용)
```

## 원본으로 실행하려면 (참가자 본인)

`data/raw/` 에 중앙회 제공 xlsx 2개를 복사하면 자동으로 원본을 우선 사용합니다.
(원본 폴더 `★중앙회 제공데이터/` 가 프로젝트 루트에 있어도 인식합니다.)
